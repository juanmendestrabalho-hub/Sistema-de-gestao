# Sistema Integrador de Marketplaces

Ferramenta para vendedores gerenciarem seus anúncios de forma centralizada.

## Funcionalidades
- Integração via API oficial com Mercado Livre.
- Sincronização automática de preços e estoques.
- Dashboard simplificado para vendedores.

## Como rodar
1. Clone o repositório.
2. No `/backend`, crie um arquivo `.env` com seu `ML_ACCESS_TOKEN`.
3. Instale as dependências: `npm install`.
4. Rode o servidor: `node server.js`.

## Tecnologias utilizadas
- **Back-end:** Node.js, Express, Axios.
- **Front-end:** React.js, CSS Grid.
